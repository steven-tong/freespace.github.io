建立文件

vim ~/.vimrc   "vim配置文件
vim ~/.vimrc.bundles    "插件配置文件
mkdir ~/.vim    "插件安装目录

插件管理工具vundle安装、配置与使用

git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle "安装vundle

使用vundle：运行vim，键入以下命令 

:PluginInstall     "安装插件 （需先在配置文件中添加插件）
:BundleClean     "卸载插件 （需先在配置文件中删除插件）
:BundleUpdate     "更新插件
:BundleList     "查看已安装插件
:BundleSearch     "搜索插件

插件配置文件(.vimrc.bundles)设置

if &compatible
  set nocompatible
end

filetype off
set rtp+=~/.vim/bundle/vundle/
call vundle#rc()

" Let Vundle manage Vundle
Bundle 'gmarik/vundle'

" Define bundles via Github repos
Bundle "tomasr/molokai"
Bundle "JamshedVesuna/vim-markdown-preview"
（继续添加你想要的插件）

if filereadable(expand("~/.vimrc.bundles.local"))
  source ~/.vimrc.bundles.local
endif

filetype on

Vim配置文件(.vimrc)设置

set nocompatible  " 去掉有关vi一致性模式，避免以前版本的一些bug和局限，必须放开头

" 启用插件配置
if filereadable(expand("~/.vimrc.bundles"))
  source ~/.vimrc.bundles
endif

" 高亮当前行列
au WinLeave * set nocursorline nocursorcolumn
au WinEnter * set cursorline cursorcolumn
set cursorline cursorcolumn

" 高亮显示第73列
set textwidth=72
set colorcolumn=+1

set number  " 显示行号
set numberwidth=5  " 行号显示宽度

set nobackup  "禁止生成临时文件
set noswapfile  "禁止生成临时文件
set nowritebackup

set backspace=2   " 像其他编辑器一样退格
set showcmd " 输入的命令显示出来，看的清楚些
set ignorecase  "  搜索忽略大小写
set nohls  " 默认情况下，寻找匹配是高亮度显示的，该设置关闭高亮显示
set incsearch  " 设置查询，如此查询时非常方便，如要查找book单词，当输入到/b时，会自动找到第一个b开头的单词
set showmatch  " 设置匹配模式，如当输入一个左括号时会匹配相应的右括号

set autowrite " 自动保存
set autoread  " 设置当文件被改动时自动载入
set confirm  " 在处理未保存或只读文件的时候，弹出确认
set clipboard+=unnamed  " 共享剪贴板
set laststatus=2  " 总是显示状态栏
set ruler  " 在编辑过程中，在右下角显示光标位置的状态行
set vb t_vb=  " 当vim进行编辑时，如果命令错误，会发出一个响声，该设置去掉响声
set mouse=a  " 设置鼠标光标可用

filetype plugin indent on  " 检测文件类型，启动插件，启动缩进
set history=1000  " 记录历史的行数
syntax on  " 语法高亮显示

set tabstop=4  " 设置tab建为4个空格
set shiftwidth=4  " 设置行之间交错时使用4个空格
set softtabstop=4  " 统一缩进为4
set autoindent  " 自动对齐，即将当前行的对齐格式应用到下一行
set smartindent  " 根据上面的对齐格式，智能的选择对齐格式

set fileencodings=utf-8,gb18030,gbk,big5,gb2312,cp936   " 中文支持

" set guifont=Courier_New:h10:cANSI    " 设置字体
" set background=dark  " 设置背景颜色
" set guioptions-=T  " 去除vim的GUI版本中的toolbar
" set guioptions-=T " 隐藏工具栏
" set guioptions-=m " 隐藏菜单栏

" 配色主题
colorscheme molokai
highlight NonText guibg=#060606
highlight Folded guibg=#0A0A0A guifg=#9090D0

Vim插件推荐

  • 安装Molokai主题

    在.vimrc.bundles中加入Bundle "tomasr/molokai"并安装

  • 安装Markdown预览

    sudo pip install grip 安装grip

    在.vimrc.bundles中加入Bundle “JamshedVesuna/vim-markdown-preview”并安装

    在.vimrc中加入 let vim_markdown_preview_github=1

    vim中按Ctrl+p即可弹出浏览器预览文档

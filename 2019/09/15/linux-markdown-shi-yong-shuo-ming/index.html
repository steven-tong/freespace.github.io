段落

  • 换行

    两个以上空格加回车

  • 字体

      普通文本
  *斜体文本*
  **粗体文本**
  ***粗斜体文本***

  • 分隔线

    三个以上*或-，中间可以有空格

      ***或* * *
  ---或- - -

  • 删除线

    文字两端加双波浪线~~即可

      ～～删除线～～

  • 下划线

    通过<u>标签来实现

      <u>下划线文本</u>

  • 脚注

    需要创建脚注的文字 [^脚注]。
    [^脚注]: 脚注内容

列表

  • 无序列表

    无序列表使用星号*、加号+或是减号-作为列表标记，例如：

      * 第一项
  * 第二项
  * 第三项

  • 有序列表

    有序列表使用数字并加上 . 号来表示， 例如：

      1. 第一项
  2. 第二项
  3. 第三项

  • 列表嵌套

    列表嵌套只需在子列表中的选项添加四个空格即可，例如：

      1. 第一项：
      - 第一项嵌套的第一个元素
      - 第一项嵌套的第二个元素
  2. 第二项：
      - 第二项嵌套的第一个元素
      - 第二项嵌套的第一个元素

区块

  • 区块使用

    区块引用是在段落开头使用>符号 ，然后后面紧跟一个空格符号，例如：

    区块文字1
区块文字2
区块文字3

  • 区块嵌套

    一个>符号是最外层，两个>符号是第一层嵌套，以此类推退，例如：

    最外层
> 第一层嵌套
> > 第二层嵌套

  • 区块使用列表

    直接使用，例如：

    区块中使用列表
1. 第一项
2. 第二项
+ 第一项
+ 第二项
+ 第三项

  • 列表中使用区块

    在>前添加四个空格的缩进，例如：

      * 第一项
区块文字1
区块文字2
  * 第二项

代码

  • 行间代码

    直接用反引号 ( ` ) 包起来，例如；

      `function`  #函数

  • 区块代码

    用 ( ``` ) 包裹一段代码，并指定一种语言（也可以不指定），例如:

    ```python
    str = input(“请输入：”)
    print (“你输入的内容是: “, str)
    ```

链接

[链接名称](链接地址)

或者

<链接地址>

图片

  • 开头一个感叹号 !
  • 接着一个方括号，里面放上图片的替代文字
  • 接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上选择性的 ‘title’ 属性的文字。
  • 例如：
      ![alt 属性文本](图片地址 "可选标题")

表格

  • 使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行，例如：
      |  表头   | 表头  |
  |  ----  | ----  |
  | 单元格  | 单元格 |
  | 单元格  | 单元格 |
  • 表格对齐
    • -: 设置内容和标题栏居右对齐。
    • :- 设置内容和标题栏居左对齐。
    • :-: 设置内容和标题栏居中对齐。
    • 例如：
      | 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
  • 注意：表格前必须以空行与前面文字隔开，否则会被当成普通文本处理

转义

使用反斜杠\转义特殊字符, 例如：\*。

公式

使用两个美元符 $$ 包裹 TeX 或 LaTeX 格式的数学公式，例如：

$$
\mathbf{V}_1 \times \mathbf{V}_2 =  \begin{vmatrix} 
\mathbf{i} & \mathbf{j} & \mathbf{k} \\
\frac{\partial X}{\partial u} &  \frac{\partial Y}{\partial u} & 0 \\
\frac{\partial X}{\partial v} &  \frac{\partial Y}{\partial v} & 0 \\
\end{vmatrix}
$$tep1}{\style{visibility:hidden}{(x+1)(x+1)}}
$$

注意：提交后需加载 Mathjax 对数学公式进行渲染。

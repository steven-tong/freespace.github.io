Fortran 语法简易说明

一般说明

  1. 字符集
    • 英文26个字母，不区分大小写
    • 数字0-9
    • 22个特殊符号:=+-*/(),.'!"%&;<>?$_空格
  2. 书面格式
    • Fortran程序以PROGRAM 程序名开始，以END结束
    • 固定格式
      • 第1个字符，C或c或*,代表此行为注释
      • 第1-5个字符，数字或空格，数字代表行号
      • 第6个字符，0以外的任何字符代表此行承接上一行
      • 第7-72个字符，代码正文
      • 第73个字符以后，无效，自动忽略
    • 自由格式
      • 叹号!以后均为注释
      • 行号直接放到最前面
      • 行尾或行首为符号&，代表此行和下一行或上一行承接
      • 每行可编写132个字符
  3. 数据类型
    • 整型
      • 长整型（4bytes）：-2147483648+2147483647， 或-2E31+1+2E31
      • 短整型（2bytes）：-32768+32767，或-2E15+12E15
    • 浮点型
      • 单精度（4bytes）：有效位数6-7位
      • 双精度（8bytes）：有效位数15-16位
    • 复数

      a+bi, 以浮点数记录实部与虚部，同样分为单精度与双精度

    • 字符（串）
    • 逻辑判断

      true(1), false(0)

  4. 数学表达式

    加（+），减（-），乘（），除（/），乘幂（*）

变量声明

  1. 整型声明

    • 长整型 
        integer(kind=4) a    !Fortran90添加
  integer*4 a   !Fortran77写法
  integer(4) a   !Fortran77写法
    • 短整型 
        integer(kind=2) a    !Fortran90添加
  integer*2 a   !Fortran77写法
  integer(2) a   !Fortran77写法

      一般情况，编译器默认使用长整型

  2. 浮点型

    • 单精度 
        real(kind=4) a    !Fortran90添加
  real*4 a   !Fortran77写法
  real(4) a   !Fortran77写法
    • 双精度 
        real(kind=8) a    !Fortran90添加
  real*8 a   !Fortran77写法
  real(8) a   !Fortran77写法
  Double Precision a    !Fortran77写法
      • 所有浮点数在计算机中均以科学计数法存储，单精度以E表示，双精度以D表示，例如3.3e0，3.5d0
      • 不指定长度默认使用单精度
      • 数字写成3.3e0或3.5d0表示强制使用单精度或双精度

  3. 复数

    • 单精度 
        complex(kind=4) a    !Fortran90添加
  complex*4 a   !Fortran77写法
  complex(4) a   !Fortran77写法
    • 双精度 
        complex(kind=8) a    !Fortran90添加
  complex*8 a   !Fortran77写法
  complex(8) a   !Fortran77写法

      赋值是使用a=(x,y), x为实部，y为虚部

  4. 字符（串）

    • 声明方式 
        character a   !不指定长度
  character(len=10) a   !Fortran90添加
  character(10) a   !Fortran77写法
  character*10 a   !Fortran77写法
  character*(10)d a   !Fortran77写法

      Fortran90赋值字符串用单引号双引号均可，Fortran77只能使用单引号。

  5. 逻辑变量

    • 长型 
        logical(kind=4) a    !Fortran90添加
  logical*4 a   !Fortran77写法
  logical(4) a   !Fortran77写法
    • 短型 
        logical(kind=2) a    !Fortran90添加
  logical*2 a   !Fortran77写法
  logical(2) a   !Fortran77写法

      一般情况，不需指定长度，有系统自动分配

  6. 自定义数据类型(Fortran90专有)

    • 定义 

        type :: person   !自定义变量类型为person
    character(len=30) :: name   !记录人名
    integer :: age   !记录年龄
    real :: length   !记录身高
  end type person   !结束定义

    • 赋值 

        type(person) :: a   !声明person类型的变量a
  a%name = "Mike"   !给元素name赋值Mike
  a%age = 30   !给元素age赋值30
  a%length = 174.5   !给元素length赋值174.5

  a=person("Mike", 30, 174.5)   !一次性给所有元素赋值

  7. 补充说明

    • 设置默认类型 

        implicit integer(a,b,c)   !a、b、c开头的变量都为整型
  implicit integer(a-f,i,k)  !a到f，以及i和k开头的变量都是整型
  implicit real(a-m)   !a到m开头的变量都是浮点型
  implicit none   !关闭默认类型
      • implicit命令必须放在所有声明之前
      • 建议在所有程序中使用implicit none命令

    • 常数声明 

        real pi
  parameter(pi=3.14)   !通用写法

  real, parameter :: pi=3.14    !Fortran90新增

    • 设置变量初值 

        integer :: a=1   !Fortran90新增

  integer b
  real c
  data b,c /2,5.6/   !传统写法

    • 等价声明 

        integer a,b
  equivalence(a,b)   !声明a与b等价，即a与b占用同一块内存

输入/输出

  1. 输入 

     read(*,*) a   !读取变量a
    • 括号内第一个参数为读取位置，星号为默认值，为键盘
    • 括号内第二个参数为读取格式，星号为默认
    • 执行完read，系统自动跳到下一行等待输入

  2. 输出 

     write(*,*) a   !输出变量a
    • 括号内第一个参数为输出位置，星号为默认值，为屏幕
    • 括号内第二个参数为输出格式，星号为默认
    • 执行完write，系统自动跳到下一行等待输出

  3. 格式化输入输出 

     write(*,100) a,b,c,d,e,f,g   !以行号为100的行中设定的格式输出变量a到屏幕
 100 format(f15.10, 2x, 3(f15.10), 2x, I4, 2x, 2(f15.10))

 write(*,"(f15.10, 2x, 3(f15.10), 2x, I4, 2x, 2(f15.10))") a   !直接将格式写在write命令里

  4. 常用输入输出格式 

     Iw[.m]   !以w个字符输出整数，至少输出m个数字，m可省略

 Fw.d   !以w个字符输出浮点数，小数部分占d个字符

 Ew.d[Ee]   !用科学计数法，以w个字符输出浮点数，小数部分占d个字符，指数部分最少输出e个数字，e可省略

 Dw.d[De]   !与Ew.d[Ee]相同，输出时代表指数的字母为D

 Aw   !以w个字符宽度输出字符串

 nX   !输出位置向右移动n位，留出n个空格

 Lw   !以w个字符输出真假值T或F

 Gw.d[Ee]   !输出所有类型数据，w，d，e含义同上，d不一定用得上，但不能省略，可以随便给个值

 /   !换行输出

 Tc   !输出位置移动到本行第c个字节

 '字符串'   !直接放入要显示的字符串

 SP/SS   !SP用来在数字前面显示正负号，SS取消SP

 BN/BZ   !输入时，BN代表没有数据的字节没有东西，BZ代表没有数据的字节为0，例如：以I5读入数字1，BN读入的数据为1，BZ读入的数据为10000

 kP   !自动为读入的浮点数乘以10的k次方

文件操作

  1. 打开文件

    Fortran可以操作文本文件与二进制文件，访问方式又顺序访问与直接访问两种 

     open(unit=number, file='filename', form='', access='', rec=length, blank='')
    • 正整数number代表文件代号，用来带起read、write命令中的输入输出位置（注意：避免使用1/2/5/6，2/6是默认输出屏幕位置，1/5是默认输入键盘位）
    • ‘filename’为打开的文件名，可以带路径
    • form代表文件类型，=‘formated’代表文本文件，=‘unformated’代表二进制文件，默认为文本文件
    • access代表文件访问方式，=‘sequential’代表顺序访问，=‘direct’代表直接访问，默认为顺序访问
    • length在顺序访问时代表一次读写数据的最大容量，在直接访问时代表每个模块的分区长度。length在文本文件下单位为1byte，在二进制格式下又编译器指定。
    • blank代表输入数字时，格式字段中的空格代表的意义，=‘null’时代表没有东西，=‘zero’时会自动以0代入。默认为=‘zero’

  2. 读写文件 

     write/read(unit=number, fmt=format, nml=namelist, rec=record, advance='')
    • number用来指定输入输出位置
    • format指定输入输出格式
    • namelist指定需要读写的namelist
    • record指定在直接访问文件时要读写的文件位置
    • advance为Fortran90新增，代表命令执行完毕后，读写位置是否自动向下移动一行，=‘yes’代表移动，为默认状态，=‘no’代表不移动

  3. NAMELIST

    NAMELIST用来把一组变量封装到一起，使用方法如下: 

     integer a,b,c
 namelist /nl_name/ a,b,c    !定义NAMELIST，必须放在声明部分
 &nl_name a=1 b=2 c=3 /   !给NAMELIST内的变量赋值
 read(10,nml='nu_name')   !读取文件内容到NAMELIST
 write(20,nml='nu_name')   !将NAMELIST内容写入到文件

流程控制与逻辑判断

  1. 语法结构

    • IF语句 

        if (Condition 1) 程序语句   !程序语句只能有一个命令

  if (Condition 1) then
      ...                !Condition 1成立时，运行此程序段
  else if (Condition 2) then
      ...                !Condition 2成立时，运行此程序段
  else if (Condition 3) then
      ...                !Condition 3成立时，运行此程序段
  else
      ...                !Condition 1、2、3都不成立时，运行此模块
  end if

    • Select语句 

        select case (variable)
  case (Num1)
      ...                !variable等于Num1时，执行此程序段
  case (Num2:Num3)
      ...                !Num2<=variable<=Num3时，执行此程序段
  case (Num4:)
      ...                !variable>=Num4时，执行此程序段
  case (:Num5)
      ...                !variable<=Num5时，执行此程序段
  case (Num6,Num7,Num8)
      ...                !variable=Num6或Num7或Num8时，执行此程序段
  case default
      ...                !variable不等于任何Num时，执行此程序段
  end select
      • case中只能使用整型，字符型和逻辑变量类型，不能使用浮点数和复数
      • case中只能使用常量，不能使用变量

    • Goto语句 

        goto 300    !跳到标号为300的行执行程序

  if (c) 10,20,30   !判断c的值，c<0,跳到第10行，c=0,跳到第20行，c>0,跳到第30行

    • Pause, Continue, Stop语句

      • 程序执行到Pause暂停，知道用户输入enter继续
      • continue没有实际意义，就是继续向下执行程序的意思
      • stop意为结束程序

  2. 逻辑判断

    • 逻辑运算

      逻辑运算 Fortran 90 Fortran 77
      相等 == .EQ.
      不相等 /= .NE.
      大于 > .GT.
      大于等于 >= .GE.
      小于 < .LT.
      小于等于 <= .LE.

    • 逻辑关系

      写法 逻辑关系
      .AND. 交集：两边都成立才成立
      .OR. 并集：两边只要有一个成立就成立
      .NOT. 逻辑反向：后面不成立才成立
      .EQV. 逻辑相同：两边相同就成立
      .NEQV. 逻辑相反：两边不相同就成立
